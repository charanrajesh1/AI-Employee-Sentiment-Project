# AI Employee Sentiment Analysis Project

## Overview
This project analyzes employee feedback to understand overall sentiment and identify potential employee flight risk. The system processes employee review data, performs sentiment classification, explores trends through data visualization, and predicts sentiment patterns using linear regression.

The project is implemented using Python and common data science libraries such as Pandas, Matplotlib, and Scikit-learn.

---

## Project Features

The system performs the following tasks:

1. **Sentiment Labeling**
   - Classifies employee reviews into:
     - Positive
     - Negative
     - Neutral

2. **Exploratory Data Analysis (EDA)**
   - Generates visualizations showing sentiment distribution.

3. **Monthly Sentiment Scoring**
   - Aggregates sentiment trends over time.

4. **Employee Ranking**
   - Ranks employees based on sentiment scores.

5. **Flight Risk Identification**
   - Detects employees likely to leave the company based on negative sentiment patterns.

6. **Linear Regression Model**
   - Predicts sentiment trends over time.
